# Alura Store

Análisis de datos de las 4 tiendas del desafío Alura Store.

## Estructura
- data/: CSV con los datos (tienda_1.csv, tienda_2.csv, ...)
- notebooks/: Notebook de análisis (analisis_alura.ipynb)
- scripts/: Scripts auxiliares en Python

## Requisitos
- Python 3.8+
- pandas, matplotlib, seaborn
Instalar: `pip install pandas matplotlib seaborn`

## Uso rápido
1. Abrir `notebooks/analisis_alura.ipynb` en Google Colab (File → Open notebook → GitHub).
2. Ejecutar las celdas para cargar `data/` desde la URL raw de GitHub o montando Google Drive.
